Override your [source](/synopkg/config/source) configuration to run <code>{`synopkg ${props.command}`}</code> on only some package.json files.

<Code
  code={`# only the root package
synopkg ${props.command} --source 'package.json'
# only packages matching a glob
synopkg ${props.command} --source 'packages/beta-*'
# multiple values can be provided
synopkg ${props.command} --source 'package.json' --source 'packages/beta-*'`}
  lang="bash"
/>

:::tip

Add quotes around your `--source` patterns so your shell does not interpret them before they're received by synopkg.

:::
